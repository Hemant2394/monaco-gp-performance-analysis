# 🏆 Monaco Grand Prix 2023 Performance Analysis

## Overview

This project provides a comprehensive performance comparison between Lewis Hamilton and Max Verstappen during the 2023 Monaco Grand Prix. The analysis combines race data with fuel consumption, tyre degradation, lap times, and temperature variation to evaluate race performance.
